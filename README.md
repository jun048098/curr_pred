# 커리큘럼으로 학과명 예측 모델
+ 충남권 ICT이노베이션 스퀘어 조성 사업의 AI 기술을 활용한 텍스트 마이닝 팀프로젝트입니다.
+ 커리큘럼을 입력하면 관련학과를 예상하여 반환합니다.

## 개발환경
+ colab으로 크롤링 및 모델링 후 파이참에서 django로 웹 구현했습니다.
+ 사용한 mecab 버전입니다.
+ https://github.com/Pusnow/mecab-ko-msvc/releases/tag/release-0.9.2-msvc-3
